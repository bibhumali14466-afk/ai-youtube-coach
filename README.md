# AI YouTube Coach

AI tool that helps YouTube creators understand why their videos are not performing well and how to improve them.

## What it does
- Analyzes why a video did not perform well
- Suggests better hooks (first 5-10 seconds)
- Recommends better titles and ideas
- Gives content improvement suggestions

## Tech Stack
- Next.js (frontend)
- Google Gemini API (AI engine)
- Supabase (database)
- Vercel (hosting)

## Goal
To build a zero-cost AI tool that helps creators improve their YouTube content with actionable feedback.
